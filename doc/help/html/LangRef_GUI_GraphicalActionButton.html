Language Reference - GUI - Graphical Action Button

A graphical button works like the Action Button but shows a custom image instead of an ordinary button.

Text Methods

N/A

Context Menu

Note: Edit is chosen automatically instead of showing a context menu with only one choice.

  • Edit: Opens the graphical action button's edit dialog.

    File:
    The image path.

    Actions:
    The actions to execute when the button is pressed. Only stored actions are allowed.

    Enabled when stopped:
    If checked the action button will be executed also when the application is stopped. (Default: Unchecked)

    Def. Size:
    The settings are saved and the size of the Icon object is changed to the size of the associated image.
