Same as Rectangle with the following modifications:

  • Default pen width: 3
  • Default pen color: Gray (#808080)

  • Another checkbox in the edit dialog:

    Cubic Bezier:
    If checked the points are interconnected with cubic Bezier splines instead of straight lines. (Default: Unchecked)
