Language Reference - GUI - Line

A Line object.

Drawing Lines

  • Start line drawing mode by clicking the Line or Spline button.
  • Add line points by clicking.
  • Exit line-drawing mode by pressing Esc or by double-clicking.
  • If the start point and the end point are sufficiently close to each other a polygon or "curved polygon" is created rather than a line or spline. The start and end points are considered sufficiently close if they have the same nearest grid point.

Context Menu

Same as Rectangle with the following modifications to the edit dialog:

  • No brush properties and fill color.

  • Added Cubic Bezier:
    If checked the points are interconnected with cubic Bezier splines instead of straight lines. (Default: Unchecked)

  • Added Arrow Properties:

    • No Arrows (Default)
    • To Arrow. An arrow at the end-point.
    • From Arrow. An arrow at the start-point.
    • Both Arrows. Arrows at both the end- and the start-point
