Editor - File Menu

  • New

    Create a new application.

  • Open

    Open a saved application.

  • Close

    Close the current workspace. If it is a top-level workspace, the entire application is closed.

  • Save

    Save the current application. The first time an application is saved you need to specify the file name.

    Note: It is not possible to save while the application is executing.

  • Save As

    The same as Save except that the file name must be specified each time.

  • OpenAll

    Open all applications in a file that has been saved using SaveAll or SaveAllAs.

  • SaveAll

    Save all applications to a single file. The first time all applications are saved you need to specify the file name.

  • SaveAllAs

    The same as SaveAll except that the file name must be specified each time.

  • Print

    Print the current workspace.

  • Exit

    Exit JGrafchart.
