Editor - Help Menu

  • Online Help

    Opens the interactive help system browser.

  • Object Help

    Changes to object help mode, in which specific help information is obtained by clicking on an object, a menu option, or a toolbar button.

  • Open Online Tutorial

    Opens the JGrafchart interactive online tutorial containing examples of how to use the different elements of JGrafchart. The tutorial is an ordinary JGrafchart application stored in doc/tutorial/tutorial.xml that is easily changed and extended.

  • Close Online Tutorial

    Closes the online tutorial.

  • About

    Opens the about dialog.
