Language Reference - GUI - Browser

The Browser object is a text string message browser that can for example be used for alarm and event displays. The messages are not persistent in the sense that they are not saved with the application. The messages are indexed so that 0 corresponds to the first message = the top of the browser.

Context Menu

  • Edit

    Opens the browser properties dialog. Changing the settings will not affect existing messages.

    Font Size:
    The font size for new messages. (Default: 12)

    Text Color:
    The text color of new messages. (Default: Black)

    Fill Color:
    The background color of the browser and of new messages. (Default: Gray (#E6E6E6))

    Font Face:
    The font face of new messages. (Default: Serif)

    Bold:
    If checked new messages are bold. (Default: Unchecked)

  • Name

    Open a dialog to set the browser's name.
