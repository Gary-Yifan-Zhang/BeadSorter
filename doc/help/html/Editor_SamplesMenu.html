Editor - Samples Menu

This menu contains various sample applications.

  • Interactive PID Basic

    Interactive PID control of the water level in a simulated water tank process. The controller gain (K), integral time (Ti), and derivative time (Td) as well as the level reference (LevelRef) can be changed during execution by editing the corresponding variable values.

    This sample also shows how the PID Procedure in the included control library can be used.
