Context Menu

  • Show/Hide Body

    Toggle the visibility of the subworkspace.

  • Table

    Opens an Inspector for the subworkspace.

  • Edit

    Opens the workspace object settings dialog.

    Relative scan period:
    The execution period of the workspace object is the execution period of the application times the realtive scan period. This is useful if the application contains parts with different execution rate demands. (Default: 1)

    Enabled:
    Decides if the workspace object is executed. If disabled it is still possible to call procedures within the workspace object. Also note that inputs within a workspace object are considered to belong to the application and will hence still be updated as usual. (Default: Checked)

    Use Icon:
    If checked, use the custom image given by Icon. (Default: Unchecked)

    Icon:
    The image to show if Use Icon is enabled. (Default: <empty>)

    Def. Size:
    The settings are saved and the size of the workspace object is changed to the size of the associated image.
