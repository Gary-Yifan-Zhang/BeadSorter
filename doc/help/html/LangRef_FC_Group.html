Language Reference - FC - Group

Note: Group is not available in the palette, it is created using Group.

A group is a collection of other objects. Groups can be objectified to hide their contents.

Hint: If a group doesn't have any empty space, its context menu can still be accessed from a Tree View.

Context Menu
