Editor - Mouse Interaction

Select

Selection works as usual, i.e. click on an object to select it, multi-select by click-dragging and include/exclude an object from the selection by pressing the Shift or Ctrl key while clicking

Move

Moving of a selection works as usual. Approaching the workspace borders will conveniently scroll the workspace.

Pressing the Ctrl key during a move or moving the selection to another workspace will instead create a copy of the selection.

Resize

  • When an object is selected it is highlighted by showing its resize handles. These can be used to resize the object.

Object Context Menu

  • Double-click or right-click on an object to show its context menu.
