Language Reference - FC - Step

A step represents a state of the application. The step can be active or inactive. An active step is indicated by a token. Step actions are shown in the action block associated with the step. The action block can be hidden (default) or shown. If the step contains actions, the action text is shown as a tool-tip text when the action block is hidden.

A step may have a name. The name can be edited by clicking on it or with Set Name in the context menu. Step names are needed in references to the .s, .t, and .x attributes of the step.

Text Methods

Common Methods

Context Menu

  • Show Action Block

    Shows the step's action block.

  • Hide Action Block

    Hides the step's action block.

  • Edit

    Opens an editor for the step's actions. The editor can also be brought up by clicking on the action text in the action block.

  • Set Name

    Open a dialog to set the step's name.
