Language Reference - IO - Analog Input

Note: To use real IO you need to disable Simulator Mode.

An analog input is a real variable that is connected to a analog input channel specified by the channel name. The channel name is usually an integer. What the analog input actually does is up to the current Custom IO.

It is not allowed to assign to an analog input from the application.
